Команды

Список доступных команд

Start Game

Команда StartGame меняет статус партии с NotStarted на InProgress.

В случает, если текущий статус отличается от NotStarted, программа уходит в панику.

Finish Game

Команда FinishGame меняет статус партии с InProgress на Finished.

В случает, если текущий статус отличается от InProgress, программа уходит в панику.

Switch Initiative

Команда SwitchInitiative меняет статус инициативы у обоих игроков.

Структура Game

Game - структура, содержащая в себе состояние конкретной партии в Response TCG и шину команд.

Состояние

Состояние партии хранится в переменной State

Статус партии

Может иметь следующие значения:

  • Не начата

  • В процессе

  • Завершена

  • Завершена программной ошибкой

Статусы двух игроков

Хранят слудующие данные:

  • наличие инициативы (есть или нет)

Шина команд

Каждое изменение состояния партии происходит с помощью команд, которые можно выполнить или отменить после выполнения.

Хранение очереди команд и их выполнение организовано с помощию структуры CommandBus.
